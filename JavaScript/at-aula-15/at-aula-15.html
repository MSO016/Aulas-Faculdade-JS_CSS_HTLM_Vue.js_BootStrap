AJAX (Asynchronous JavaScript and XML)


Definição:

AJAX é um conjunto de técnicas de desenvolvimento web que permite que páginas da web se comuniquem com servidores de forma assíncrona, sem precisar recarregar a página inteira. O nome vem de "JavaScript e XML assíncronos", embora hoje em dia o formato JSON seja mais utilizado no lugar do XML.

Características:

  • Permite atualizar partes específicas da página sem recarregar tudo.
  • Utiliza JavaScript para fazer requisições ao servidor.
  • Comumente usa JSON para trocar dados, embora suporte XML e outros formatos.
  • Melhora a experiência do usuário, tornando a navegação mais rápida e fluida.

Propósito:

O principal propósito do AJAX é tornar as aplicações web mais interativas e responsivas. Ele é muito usado em formulários, carregamento de listas dinâmicas, sistemas de busca em tempo real e redes sociais, onde o conteúdo muda sem recarregar toda a página.
